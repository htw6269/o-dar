## 설치

1. 프로젝트를 클론합니다.
   ```bash
   git clone https://github.com/사용자명/프로젝트명.git
   ```
2. 프로젝트 디렉토리로 이동합니다.
   ```bash
   cd 프로젝트명
   ```
3. 필요한 종속성 패키지들을 설치합니다.
   ```bash
   npm install
   ```

## 사용법

프로젝트를 로컬 환경에서 실행하려면 다음 명령을 실행합니다.
```bash
npm start
```
이후에 [http://localhost:3000](http://localhost:3000) 에서 프로젝트를 확인할 수 있습니다.
